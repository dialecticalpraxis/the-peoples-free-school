# Readings by Author

An alphabetical index of authors whose work is drawn on across the school's themes. Cross-references point to the theme and lens where each work appears.

This index is in early development. As resources are added to themes, they should also be indexed here.

## How to use this index

Each entry follows this format:

```
**Author Name** (years lived)
- *Work Title* (publication year). Brief note on the work and what it offers.
  - Used in: [[theme-name#lens|Theme name, Lens]]
```

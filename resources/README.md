# Resources

This folder holds cross-cutting indexes of materials used across the school's themes. The primary location for materials is within each theme folder (organized by lens). These indexes are alternate views that let you find material by author, format, or other criteria when that is useful.

## Indexes

- [[readings-by-author|Readings by author]]: alphabetical index of authors and the works of theirs the school draws on
- [[podcasts|Podcasts]]: ongoing podcasts and one-off episodes worth knowing about
- [[films|Films and documentaries]]: feature-length films, documentaries, and short films
- [[primary-sources|Primary sources]]: speeches, manifestos, documents, archives, oral histories

## How to add resources

Add resources first to the relevant theme and lens (e.g., a Federici reading goes in `themes/04-gender-reproduction-and-the-household/historical.md`). Then add a cross-reference to the appropriate index here.

This avoids duplication while letting users find things by either path.

See the [contributing guide](../CONTRIBUTING.md) for the contribution process.

## A note on access

Many of the resources the school uses are available freely online. Others require library access, purchase, or subscription. The school does not host or distribute copyrighted material. When a reading is paywalled or only available in print, the school links to a publisher page and notes the situation.

For groups in places without good library access, consider building relationships with sympathetic librarians, university interlibrary loan systems, and free online archives (the Internet Archive, the Marxists Internet Archive, the Anarchist Library). Mutual aid for book access is a part of how the school sustains itself.
